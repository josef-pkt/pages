circularstats.api.circ_kurtosis¶

static api.circ_kurtosis(alpha, w=1, axis=0)¶

Compute measures of angular kurtosis.

Parameters :

alpha : array_like

data, sample of angles in radians

w : float or ndarray

case weight, number of observations in case of binned angle data

axis : int or None

Axis over which the mean resultant is calculated, default is 0.
Returns :

k : float or ndarray

kurtosis (from Pewsey)

k0 : float or ndarray

kurtosis (from Fisher)

References

Pewsey, Metrika, 2004

Fisher, Circular Statistics, p. 34

Philipp Berens, 2009, Circular Statistics Toolbox for Matlab
