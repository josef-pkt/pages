circularstats.api.circ_median¶

static api.circ_median(alpha, axis=0)¶

Compute the median direction for circular data.

Parameters :

alpha : array_like

data, sample of angles in radians

axis : int or None

Axis over which the mean resultant is calculated, default is 0. Must be either 0 or 1 for circ_median
Returns :

mu : float

median direction

Notes

circ_median can be slow for large datasets

iterates over each row or columns, not vectorized because it uses already a (nobs, nobs) distance matrix

References

Biostatistical Analysis, J. H. Zar (26.6)

Philipp Berens, 2009, Circular Statistics Toolbox for Matlab
