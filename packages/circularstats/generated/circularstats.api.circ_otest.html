circularstats.api.circ_otest¶

static api.circ_otest(alpha, sz=None, w=1)¶

Omnibus or Hodges-Ajne test for uniform distribution of circular data.

H0: the population is uniformly distributed around the circle
HA: the population is not distributed uniformly around the circle

Alternative to the Rayleigh and Rao’s test. Works well for unimodal, bimodal or multimodal data. If requirements of the Rayleigh test are met, the latter is more powerful.

Parameters :

alpha : array_like

data, sample of angles in radians

sz : float

step size for evaluating distribution, default 1 degree

w : float or ndarray

case weight, number of observations in case of binned angle data
Returns :

pval : float

p-value

m : int

minimum number of samples falling in one half of the circle

References

Biostatistical Analysis, J. H. Zar

A bivariate sign test, J. L. Hodges et al., 1955

A simple test for uniformity of a circular distribution, B. Ajne, 1968

Philipp Berens, 2009, Circular Statistics Toolbox for Matlab
