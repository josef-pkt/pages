circularstats.api.circ_skewness¶

static api.circ_skewness(alpha, w=1.0, axis=0)¶

Compute measures of angular skewness.

Parameters :

alpha : array_like

data, sample of angles in radians

w : float or ndarray

case weight, number of observations in case of binned angle data

axis : int or None

Axis over which the mean resultant is calculated, default is 0.
Returns :

b : float or ndarray

skewness (from Pewsey)

b0 : float or ndarray

alternative skewness measure (from Fisher)

References

Pewsey, Metrika, 2004

Statistical analysis of circular data, Fisher, p. 34

Philipp Berens, 2009, Circular Statistics Toolbox for Matlab
