circularstats.api.circ_axialmean¶

static api.circ_axialmean(alpha, m=2, axis=0)¶

Compute the mean direction for circular data with axial correction.

Parameters :

alpha : array_like

data, sample of angles in radians

m : int

repeat factor, number of repeats per unit circle

axis : int or None

Axis over which the mean resultant is calculated, default is 0.
Returns :

r mean resultant length :

mu mean direction :
