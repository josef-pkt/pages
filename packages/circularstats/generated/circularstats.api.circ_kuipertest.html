circularstats.api.circ_kuipertest¶

static api.circ_kuipertest(alpha1, alpha2, res=100, vis_on=False)¶

Kuiper two-sample test for equality of distributions

The difference can be in any property, such as mean location and dispersion. It is a circular analogue of the Kolmogorov-Smirnov test.:

H0: The two distributions are identical.
HA: The two distributions are different.
Parameters :

alpha1 : array_like

fist sample (in radians)

alpha2 : array_like

second sample (in radians)

res : float

resolution at which the cdf is evaluated

vis_on : bool

display graph NotImplemented
Returns :

pval : float

p-value the smallest of .10, .05, .02, .01, .005, .002, .001, for which the test statistic is still higher than the respective critical value. This is due to the use of tabulated values. If p>0.1, pval is set to 1. TODO: set to 0.5 instead

k : float

test statistic

K : float

critical value

References

Batschelet, 1980, p. 112

Marc J. Velasco and Philipp Berens, 2009, Circular Statistics Toolbox for Matlab
