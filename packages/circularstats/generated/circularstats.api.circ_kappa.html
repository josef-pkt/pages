circularstats.api.circ_kappa¶

static api.circ_kappa(alpha, w=1, method='approx')¶

Estimate kappa of the von Mises distribution

Computes an approximation to the ML estimate of the concentration parameter kappa of the von Mises distribution.

Parameters :

alpha : array_like

angles in radians OR alpha is length resultant

w : float or array_like

number of incidences in case of binned angle data

method : string in [‘approx’, ‘mle’, ‘ub’]

If method is “approx” (default), then an approximation is used that does not require a nonlinear fixed point search. ‘mle’ calculates the biased maximum likelihood estimate ‘ub’ calculates kappa based on unbiased root TODO: mle, ub not verified yet
Returns :

kappa : float

estimated value of kappa

References

Statistical analysis of circular data, Fisher, equation p. 88
