circularstats.api.circ_ktest¶

static api.circ_ktest(alpha1, alpha2)¶

A parametric two-sample test for equality of two concentration parameters.

H0: The two concentration parameters are equal.
HA: The two concentration parameters are different.
Parameters :

alpha1, alpha2 : array_like

data, two samples of angles in radians
Returns :

pval : float

p-value that samples have different concentrations

f : float

f-statistic calculated

Notes

Assumptions: both samples are drawn from von Mises type distributions
and their joint resultant vector length should be > 0.7

prints warning in mean resultant < 0.7 TODO make into UserWarning

References

Batschelet, 1980, section 6.9, pg 122-124

Marc J. Velasco, 2009, Circular Statistics Toolbox for Matlab
