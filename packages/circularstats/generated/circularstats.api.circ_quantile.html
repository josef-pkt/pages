circularstats.api.circ_quantile¶

static api.circ_quantile(alpha, p, mu=None)¶

quantiles of distribution centered at mu

Note I made this up in analogy and similar to bootstrap on a circle. I still need to look up an “official” definition.

open: for very asymmetric, skewed distributions this will go the shorter distance, in the bootstrap description the split is by count. BUG: this shouldn’t work with support around the entire circle and p close to one or zero.

might not work correctly for 2-Dim arrays
