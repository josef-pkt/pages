circularstats.distributions.chf_stable¶

static distributions.chf_stable(t, a, lambd, gamma, mu)¶

characteristic function of stable distribution

Parameters :

t : array_like

points at which characteristic function is evaluated

a, lambd, gamma : float (or array_like ?)

parameters of stable distribution, see Notes for examples - TODO

mu : float (or array_like ?)

mean or location parameter of underlying stable distribution
Returns :

chfval : ndarray

characteristic function evaluated at x

Notes

Can be used for higher dimensional arguments if t, mu and a broadcast.

Cauchy and Normal distributions are special cases of the stable distribution, with parameters

normal :
cauchy :

tested for the two special cases.
