circularstats.distributions.pdf_wrapped_from_charfunc¶

static distributions.pdf_wrapped_from_charfunc(chfunc, args=(), k_comp=10)¶

returns a function that is the density of a wrapped distribution

The density is defined by the Fourier series with k_comp terms.

Parameters :

chfunc : callable

characteristic function of the underlying distribution. The function is called with chfunc(t, *args)

args : tuple

arguments to use when evaluating chfunc

k_comp : int

number of terms (including zero term) to include in the series representation
Returns :

pdf_func : function

pdf_func(x) evaluates the density at points x

(coeff_cos, coeff_sin) : tuple of ndarrays

coefficients used for the Fourier series approximation
