circularstats.api.Circular.wrap¶

Circular.wrap(func, x, *func_args, **func_kwds)¶

call function func for radians with conversion

Parameters :

func : callable

function that requires radians as input and returns radians units.

x : array_like

values at which func is evaluated. x is converted to radians before calling func
Returns :

ret : array_like

values returned from func, converted to original units

Notes

This is just wrapping the func call with conversion to_radians for input and from_radians of output. Users have to check that the conversion makes sense.

Examples

>>> clock24 = Circular(low=0, upp=24, units='hours', style='clock')
>>> clock24.wrap(cs.circ_mean, data.arrival)
17.257916921461366
>>> clock24.from_radians(cs.circ_mean(clock24.to_radians(data.arrival)))
17.257916921461366
