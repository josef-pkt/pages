circularstats.distributions.circ_moments_from_pdf¶

static distributions.circ_moments_from_pdf(pdf_func, k_comp=10, low=0, upp=6.2831853071795862)¶

calculate circular moments for a distribution given by its density

Parameters :

pdf_func : callable

The density function of the distribution. Currently can take only one argument, the values at which the pdf is evaluated. (Currently no *args)

k_comp : int

number of moments to calculate

low : float

lower integration limit

upp : float

upper integration limit
Returns :

coef_cos : list

real (cosine) part of the circular moment

coef_sin : list

imaginary (sine) part of the circular moment

Notes

This function uses scipy.integrate.quad for integration. cosine and sine parts are separately integrated.
