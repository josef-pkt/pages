circularstats.api.circ_wwtest¶

static api.circ_wwtest(alpha, groups, w=1)¶

Parametric Watson-Williams multi-sample test for equal means.

Can be used as a one-way ANOVA test for circular data.

H0: all populations have equal means
HA: at least one populations has different mean

Note: Use with binned data is only advisable if binning is finer than 10 deg. In this case, alpha is assumed to correspond to bin centers.

The Watson-Williams two-sample test assumes underlying von-Mises distributrions. All groups are assumed to have a common concentration parameter k.

Parameters :

alpha : array_like

data, sample of angles in radians

idx : array_like

indicates which population the respective angle in alpha comes from.

w : float or ndarray

case weight, number of observations in case of binned angle data
Returns :

pval : float

p-value of the Watson-Williams multi-sample test. Reject H0 if p-value is small.

table : pandas.DataFrame instance

data frame containg the ANOVA table

References

Biostatistical Analysis, J. H. Zar

Philipp Berens, 2009, Circular Statistics Toolbox for Matlab
