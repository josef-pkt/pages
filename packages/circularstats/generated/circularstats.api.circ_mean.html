circularstats.api.circ_mean¶

static api.circ_mean(alpha, w=1.0, return_ci=False, xi=0.050000000000000003, axis=0)¶

Compute the mean direction for circular data.

Parameters :

alpha : array_like

data, sample of angles in radians

w : float or ndarray

case weight, number of observations in case of binned angle data

return_ci : bool

If true, then the confidence interval is also returned

xi : float

(1-xi)-confidence limits are computed, default 0.05. Only used if return_ci is true.

axis : int or None

Axis over which the mean resultant is calculated, default is 0
Returns :

mu : ndarray

mean direction

ul : ndarray

upper 95% confidence limit

ll : ndarray

lower 95% confidence limit

Notes

Note: upper is before lower confidence limit in return TODO: reverse ul,ll

Statistical analysis of circular data, N. I. Fisher
