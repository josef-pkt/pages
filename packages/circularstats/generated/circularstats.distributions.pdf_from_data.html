circularstats.distributions.pdf_from_data¶

static distributions.pdf_from_data(data, k_comp=10)¶

get a pdf function for trigonometric series density

Parameters :

data : ndarray

data given as a 1d array (TODO: check if 2-D works)

k_comp : int

number of terms in the series including zero term.
Returns :

func : function

a function that evaluates the density for given values

(coeff_cos, coeff_sin) : tuple of lists

coefficients of circular moments of the data, with zero element set to 0.5. If all coefficients of coeff_sin, are less than 1e-14, then coeff_sin is set to None.
