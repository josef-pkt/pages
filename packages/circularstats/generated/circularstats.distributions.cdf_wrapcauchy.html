circularstats.distributions.cdf_wrapcauchy¶

static distributions.cdf_wrapcauchy(x, mean, a, k_comp=10)¶

cdf of wrapped cauchy distribution

based on Fourier series expansion

Parameters :

x : array_like

points at which density is evaluated

mean : float (or array_like ?)

mean of underlying normal distribution

a : float (or array_like ?)

parameter of underlying cauchy distribution

k_comp : int

number of terms (including zero term) to include in the series representation
Returns :

cdfval : ndarray

cumulative density evaluated at x The cdf is normalized with the first value of x as the zero value.
