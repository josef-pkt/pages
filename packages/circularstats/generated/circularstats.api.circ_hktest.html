circularstats.api.circ_hktest¶

static api.circ_hktest(alpha, idp, idq, inter=True, fn=None)¶

Parametric two-way ANOVA for circular data with interations.

Parameters :

alpha : array_like

data, sample of angles in radians

idp : array_like

levels of factor 1 used in the test

idq : array_like

levels of factor 2 used in the test

inter : bool

If true (default), interaction effects are included. If false, only main effects are included

fn : list

list containing strings with the names of the factors
Returns :

pval : list

pvalues for column, row and interaction effects

table : list

list containg the anova table TODO: convert to pandas DataFrame

Notes

The test assumes underlying von-Mises distributrions.

All groups are assumed to have a common concentration parameter k, between 0 and 2.

References

Harrison, D. and Kanji, G. K. (1988). The development of analysis of variance for circular data. Journal of applied statistics, 15(2), 197-223.

Philipp Berens, 2009, Circular Statistics Toolbox for Matlab
