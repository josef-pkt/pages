circularstats.api.circ_moment¶

static api.circ_moment(alpha, w=1, p=1, cent=False, axis=0)¶

Compute the complex p-th centered or non-centered circular moment.

Parameters :

alpha : array_like

data, sample of angles in radians

w : float or ndarray

case weight, number of observations in case of binned angle data

p : int

order of the moment, p-th moment will be computed, default is p=1.

cent : bool

If false (default), then the raw, uncentered moment will be returned. If true then the centered moment will be returned.

axis : int or None

Axis over which the mean resultant is calculated, default is 0.
Returns :

mp : float or ndarray

complex p-th moment

rho_p : float or ndarray

magnitude of the p-th moment

mu_p : float or ndarray

angle of th p-th moment

References

Statistical analysis of circular data, Fisher, p. 33/34

Philipp Berens, 2009, Circular Statistics Toolbox for Matlab
