circularstats.api.circ_cmtest¶

static api.circ_cmtest(alpha, groups)¶

Nonparametric multi-sample test for equal medians.

Similar to a Kruskal-Wallis test for linear data.

H0: the s populations have equal medians
HA: the s populations have unequal medians
Parameters :

alpha: array_like :

data, sample of angles in radians

groups : array_like

indicates which population the respective angle in alpha comes from, range(s)
Returns :

pval : float or ndarray

p-value of the common median multi-sample test. Discard H0 if

pval is small.

med : float

best estimate of shared population median if H0 is not

discarded at the 0.05 level and NaN otherwise.

P : float

test statistic of the common median test.

Notes

Loops over all groups

References

Fisher NI, 1995

Philipp Berens, 2009, Circular Statistics Toolbox for Matlab
