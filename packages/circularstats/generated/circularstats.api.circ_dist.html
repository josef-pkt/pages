circularstats.api.circ_dist¶

static api.circ_dist(x, y, pairwise=False)¶

Compute pairwise distance, difference x_i-y_i, around the circle

Parameters :

alpha : array_like

sample of linear random variable, JP: why linear?

beta : array_like

sample of linear random variable or one single angle

pairwise : bool

If pairwise is false, then it is assumed that x and y broadcast and the difference is calculated directly between x and y. If pairwise is true, then the differences are calculated between all pairs (x_i, y_i) of elements in x and in y.
Returns :

r : ndarray

matrix with distances or differences

References

Biostatistical Analysis, J. H. Zar, p. 651

Philipp Berens, 2009, Circular Statistics Toolbox for Matlab
