circularstats.api.circ_clust¶

static api.circ_clust(alpha, numclust=5, disp=0)¶

Perform a simple agglomerative clustering of angular data.

Parameters :

alpha : array_like

data, sample of angles in radians

numclust : int

number of clusters desired, default: 2

disp : bool

show plot at each step, default: false TODO: NotImplemented
Returns :

cid : ndarray, int

cluster id for each entry of alpha

alpha : ndarray

sorted angles, matched with cid

mu : ndarray

mean direction of angles in each cluster
